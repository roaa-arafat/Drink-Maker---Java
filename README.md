# Drink-Maker---Java
A simple Java project to practice OOP concepts.

## Description
The project creates a general class for making drinks, then extends it to make tea ,coffee and Herbs.

## Used Concepts
- Abstract Class
- Inheritance
- Method Overriding

## Classes
- Drink_Maker
- Tea_Maker
- Coffee_Maker
- Herbs_Maker
- Test_Maker
